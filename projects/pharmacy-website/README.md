# نظام إدارة الصيدلية المتكامل
## Pharmacy Management System

نظام متكامل لإدارة الصيدليات يعمل بالكامل على المتصفح بدون الحاجة لخادم (Serverless).

---

## 📋 المميزات الرئيسية

### 1. لوحة التحكم (Dashboard)
- إحصائيات سريعة (إجمالي الأدوية، مبيعات اليوم، منتهية الصلاحية، منخفضة المخزون)
- تنبيهات فورية للأدوية منتهية الصلاحية والمنخفضة المخزون
- رسم بياني للمبيعات
- آخر المبيعات
- أدوية منخفضة المخزون

### 2. إدارة الأدوية
- إضافة/تعديل/حذف الأدوية
- البحث باسم الدواء أو المادة الفعالة أو الباركود
- تصفية حسب التصنيف والحالة
- تتبع الصلاحية تلقائياً
- تنبيهات للأدوية منتهية وقريبة من الانتهاء
- تصدير إلى CSV

### 3. المبيعات
- إنشاء فواتير بيع جديدة
- البحث عن الأدوية باسم أو باركود
- إضافة خصم وضريبة
- حساب الإجمالي تلقائياً
- طباعة الفاتورة
- سجل كامل للمبيعات
- إلغاء الفاتورة مع إعادة الكمية للمخزون

### 4. العملاء
- إدارة بيانات العملاء
- تتبع مشتريات كل عميل

### 5. الموردين
- إدارة بيانات الموردين
- تتبع فواتير الشراء لكل مورد

### 6. المشتريات
- إنشاء فواتير شراء
- إضافة أدوية مع سعر الشراء
- تتبع المبالغ المدفوعة والمتبقية
- تحديث المخزون تلقائياً

### 7. التقارير
- تقرير المبيعات اليومية
- رسم بياني للمبيعات
- توزيع المبيعات حسب التصنيف
- أكثر الأدوية مبيعاً
- الأدوية منتهية الصلاحية
- تصفية حسب التاريخ
- تصدير التقارير

### 8. الإعدادات
- بيانات الصيدلية
- الإعدادات المالية (الضريبة، العملة)
- إعدادات التنبيهات
- إعدادات الفاتورة
- إدارة المستخدمين والصلاحيات
- إعادة تعيين البيانات

## 🚀 كيفية التشغيل

### الطريقة 1: فتح مباشر
1. قم بفتح ملف `login.html` في أي متصفح حديث
2. استخدم أحد الحسابات التجريبية:

| الدور | البريد | كلمة المرور |
|-------|--------|------------|
| صاحب الصيدلية | admin@pharmacy.com | admin123 |
| الصيدلي | pharmacist@pharmacy.com | pharma123 |
| أمين المخزن | stock@pharmacy.com | stock123 |

### الطريقة 2: خادم محلي
```bash
# Python 3
python -m http.server 8000

# Node.js
npx serve .

# PHP
php -S localhost:8000
```

ثم افتح: `http://localhost:8000/login.html`
