# أداة رفع الملفات إلى GitHub

أداة ويب تمكنك من رفع الملفات مباشرة إلى مستودع GitHub الخاص بك.

## كيفية الاستخدام

1. انتقل إلى [الصفحة الرئيسية](https://osama2020almorisi.github.io/fileuploader/)
2. اسحب وأسقط الملفات أو انقر على "اختر الملفات"
3. حدد المجلد الهدف (اختياري)
4. انقر على "رفع الملفات"
5. أدخل توكن GitHub الخاص بك عند الطلب

## متطلبات

- توكن GitHub مع صلاحيات `repo` و `workflow`

## كيفية التطوير

```bash
git clone https://github.com/osama2020almorisi/fileuploader.git
cd fileuploader
```
